# ArenaMax Gaming - Esports Community Demo

Interactive prototype demonstrating community features for esports betting platforms.

## Features

- **Live Bet Feed** - Real-time community betting activity
- **Big Wins** - Showcase of successful bets
- **Leaderboard** - Top community performers
- **Competitions** - Community prediction contests
- **Expert Analyses** - In-depth match breakdowns from analysts

## Featured Games

- League of Legends (LEC, LCK, LCS, LPL)
- Counter-Strike 2 (BLAST Premier)
- Dota 2 (The International)
- Valorant, Overwatch 2, Rocket League

## View Demo

Open `index.html` in a browser or deploy to any static hosting service.
